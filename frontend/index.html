How It Works

Four simple steps powered by our Java Spring Boot backend.

1

Upload Your Resume

Upload a PDF, DOCX, or plain text file. The file is securely sent to our Java REST endpoint.

2

Java Backend Parses the Content

Apache Tika and NLP pipelines extract skills, experience, education, and keywords.

3

AI Engine Scores & Analyzes

Parsed data is scored against ATS rules and keyword libraries stored in PostgreSQL.

4

Get Your Detailed Report

Backend returns a structured report with scores, matched keywords, and suggestions.
